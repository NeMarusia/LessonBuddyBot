# LessonBuddyBot

Telegram-бот для заявок на замены преподавателей: собирает дату, причину и группу, сохраняет заявку в SQLite и отправляет уведомление куратору.

## Возможности

- сбор заявок на замену от преподавателя;
- хранение статусов замен в SQLite;
- уведомление куратора о новой заявке;
- просмотр статусов и списка замен.

## Настройка

1. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

2. Задайте переменные окружения:

   ```bash
   export TELEGRAM_BOT_TOKEN="your_bot_token_here"
   export CURATOR_ID="143612737"
   export DB_PATH="db.sqlite3"
   ```

   Можно скопировать `.env.example` в `.env`, если запускаете проект через окружение, которое читает `.env`.

3. Запустите бота:

   ```bash
   python main.py
   ```

## Безопасность

- `Token.txt`, `.env`, локальная база SQLite и IDE-файлы не должны попадать в Git.
- Если токен бота уже был опубликован, его нужно перевыпустить у BotFather.

## Схема БД

Таблица `swaps` хранит группу, дату занятия, причину, пользователя, статус и служебные даты создания/обновления.
